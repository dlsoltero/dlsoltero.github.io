Selected Projects

MODIS: Multi-Omics Data Integration for Small and unpaired datasets

We solve a long-standing problem in the field: how to perform data integration for small unpaired datasets. This work was motivated by settings with scarce data such as rare diseases. We learn multiple Variational Autoencoders with an auxiliary adversarial modality discriminator and a class discriminator operating in the latent space. We made it easy to train and stable, by adapting the relativistic GAN loss and introduced multiple additional innovations to improve accuracy.
